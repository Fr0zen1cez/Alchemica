# ⚗️ Alchemica

A browser-based element-combination crafting game powered by AI. Start with four basic elements and combine them to discover an endless number of new ones. Every result is generated live — there are no pre-set answers.

---

## ✨ Features

### Core Gameplay
- **AI-powered combinations** — every result is generated by Gemini via Pollinations AI, no hardcoded answers
- **10 built-in worlds** — Origins, Mythology, Medieval, Biology, Space Age, Digital, Ocean Depths, Arcane, Ancient Egypt, Apocalypse — each with unique starters and a 5-item quest chain
- **AI world generation** — generate 9 brand-new expansion worlds with custom themes, starters, and quests on demand
- **Holiday worlds** — 20+ seasonal worlds (Halloween, Christmas, Lunar New Year, Diwali, and more) accessible year-round
- **Secret Void World** — unlocked by discovering `void`, `nothing`, or related concepts

### Progression
- **Account system** — username, avatar color, bio, XP, and levels (1–100)
- **XP & leveling** — earn XP per discovery based on rarity; common → 1 XP up to transcendent → 150 XP
- **Achievements** — 146 achievements across 5 tabs, including 49 pop culture references
- **Speedrun mode** — race to craft a target element as fast as possible
- **Weekly challenges** — 5 fresh discovery targets every week, seeded by calendar week
- **Weekly themed challenges** — 50 rotating themes with forbidden-tag constraints
- **Daily combo & challenge** — a new suggested combo and discovery target every day
- **Streaks** — daily and weekly login streaks with milestone badges

### Save System
- **5 save slots** — fully independent playthroughs with rename, duplicate, and delete
- **Per-world data** — items and combinations stored separately per world
- **Auto-backup** — rolling backups every 25 writes, max 3 per slot; restore from the Backups panel
- **Export / Import** — save as `.alc` file or share a compact share code
- **Sync mode** — optionally mirror discoveries across all worlds
- **Mirror folder** — auto-copy saves to any local folder (e.g. a Google Drive sync folder)

### Social & Sharing
- **Community database** — share discoveries with friends via a self-hosted webhook server or Telegram bot
- **Leaderboard** — push your stats (total discoveries, rarest item, speedrun best) to a community leaderboard
- **Gift system** — encode any item as a shareable `ALCGIFT-` code; recipients claim it directly into their save
- **Share codes** — compress your full discovery set into a single string to share with anyone

### Organisation
- **Collections** — create labeled, colored groups and drag items into them
- **Boards** — named canvas layouts for arranging items spatially
- **Item editor** — edit emoji, display name, rarity, notes, tags, and lore on any item
- **Trash & restore** — soft-delete items with single-step recovery; empty trash permanently
- **Recipe path** — trace any item back to its starter ingredients step-by-step

### Customisation
- **Themes** — built-in themes plus earnable holiday themes (complete a holiday world's quest to unlock its theme)
- **Custom themes** — import community-made or supporter theme packs (`.zip` or `.json`)
- **Layout overrides** — custom themes can change card radius, button radius, sidebar width, navbar style, font size, and more — full redesigns are possible
- **Background animations** — 11 free animations + 4 unlockable via GitHub star; custom animations via theme packs
- **Cursor trails** — 4 free trails (Sparkle Dust, Comet Tail, Soap Bubbles, Silk Ribbon) + 2 unlockable via GitHub star (Fire Trail, Galaxy Swirl)
- **Custom background** — upload an image or paste base64
- **Plugin system** — drop a `.py` file into `plugins/` to add hardcoded combinations or extra starting items
- **Custom AI endpoint** — use any OpenAI-compatible API (OpenAI, local Ollama, etc.) instead of Pollinations
- **Custom system prompts** — override the combine or tags prompts directly in Settings

### Deployment
- **Desktop app** — native window via pywebview (uses system WebView2 on Windows — no bundled Chromium, no flicker)
- **Server mode** — bind to `0.0.0.0` for LAN or internet hosting, with configurable port and custom public URL
- **Dev mode** — inject combinations manually, run stress tests

## 🚀 Quick Start

### Run from source
```bash
pip install flask requests
python app.py
# Open http://localhost:5000
```

### Desktop app
```bash
pip install flask requests pywebview pystray pillow
python desktop_app.py
```

### Build Windows EXE
```bash
pip install pyinstaller pywebview pystray pillow
python build_game.py
# Output: dist/Alchemica_win.zip
```

## 🔑 API Key

Alchemica works **free** via [Pollinations AI](https://pollinations.ai).

To use a different model or provider, enable **Custom Endpoint** in Settings → AI and point it at any OpenAI-compatible `/v1/chat/completions` endpoint.

## 🔌 Plugins

Drop a `.py` file into the `plugins/` folder:

```python
PLUGIN_NAME    = "My Plugin"
PLUGIN_VERSION = "1.0"

EXTRA_STARTING_ITEMS = []  # optional extra starters

def custom_combinations():
    return {
        "fire+water": {"result": "steam", "emoji": "♨️", "rarity": "common"},
    }

def on_combination(item_a, item_b, result):
    pass  # called after every combination
```

Enable plugins in **Settings → Advanced → Plugins**.

## 🌐 Community Database

Share discoveries with friends so nobody re-runs AI on the same combination twice.

**Webhook (recommended):**
1. Deploy `server/` to [Render](https://render.com) for free
2. In Alchemica: Settings → Community → paste your server URL → Test Connection → Save

**Telegram only (no server needed):**
1. Create a Telegram bot via @BotFather and a private channel
2. In Alchemica: Settings → Community → select Telegram backend → paste bot token + channel ID
