📄 How to Write Plugins

Drop a .py file into the plugins/ folder next to the game. It auto-loads on next refresh.

Required fields: 
PLUGIN_NAME    = "My Plugin"
PLUGIN_VERSION = "1.0"
Add custom combinations: 
def custom_combinations():
    return {
        "fire+water": {
            "result": "steam",
            "emoji": "♨️",
            "rarity": "common"
        },
        "earth+water": {
            "result": "mud",
            "emoji": "🟫",
            "rarity": "common"
        },
    }
Add extra starting items: 
EXTRA_STARTING_ITEMS = [
    {"key": "void", "emoji": "⬛",
     "display": "Void", "rarity": "rare"},
]
React to combinations: 
def on_combination(item_a, item_b, result):
    print(f"{item_a} + {item_b} = {result}")

Combination keys are always alphabetically sorted: fire+water not water+fire.
Rarity values: common, uncommon, rare, legendary.
